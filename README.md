# Kundan Kumar Singh — Portfolio

Personal portfolio site hosted on GitHub Pages: [kundan-kumar-singh.github.io](https://kundan-kumar-singh.github.io/)

## Local development

```bash
npm install
gulp watch   # recompiles SCSS and JS on change
```

Open `index.html` in a browser or use a local static server.

### Custom profile photo

Add your photo at `images/profile.jpg` (square, ~400×400px). Until then, your GitHub avatar is used automatically.

### Resume

The downloadable resume lives at `assets/kundan_resume.pdf`. Replace that file to update the site and download link.

## Stack

- HTML / CSS / JavaScript
- Bootstrap grid
- Font Awesome icons
- Formspree contact form
